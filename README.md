# 🌳 Little English Forest · 小小英语森林

> **让每一个孩子，都拥有一位永远耐心、永远鼓励、永远陪伴成长的 AI 英语老师。**

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

## ✨ 功能

- 📅 **365天课程体系** — 每天3个单词，循序渐进
- 🎤 **发音跟读** — 浏览器原生语音合成，随时开口练
- 📚 **AI绘本** — 配音绘本阅读系统
- 🌳 **成长树** — 可视化学习进度与徽章系统
- ❤️ **爸爸陪伴模式** — 每日定制亲子互动提示
- 📔 **AI成长日记** — 自动记录每天的学习亮点
- 👨‍👧 **家长后台** — 完整的学习数据面板
- 📱 **PWA** — 可安装到手机，离线可用

## 🚀 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 🛠️ 技术栈

- **Next.js 15** · App Router
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Web Speech API**
- **PWA**
